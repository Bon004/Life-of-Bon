Instead of manually reporting broken buttons and waiting for investigation, deploy an agent that reproduces bugs in a headless browser, writes a failing test, fixes the code, and verifies the fix — all without intervention. Fix the code, re-run the test until it passes, and run the full test suite to check for regressions. Only stop when all tests pass.
